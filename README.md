# Информационная система проката автомобилей

Курсовой проект по разработке веб‑приложения для автоматизации бизнес‑процессов компании по прокату автомобилей.  
Система включает серверную часть (API) на FastAPI + SQLModel + PostgreSQL и клиентский веб‑интерфейс на чистом JavaScript.

---

## Основной функционал

- Учет моделей и автомобилей автопарка.
- Ведение базы клиентов и сотрудников.
- Оформление и сопровождение заказов аренды.
- Учет платежей, штрафов, ремонтов и страховых полисов.
- Хранение отзывов по завершенным заказам.
- Цветовая индикация статусов (доступен, в аренде, оплачен, уволен и т.д.).
- Drill‑down навигация: просмотр связанных записей (заказы клиента, ремонты авто, платежи и штрафы по заказу).
- Защита целостности данных: запрет удаления записей, к которым есть связанные данные.

## Установка и запуск

### 1. Установка зависимостей

pip install fastapi uvicorn 
pip install sqlmodel psycopg2-binary

### 3. Настройка базы данных PostgreSQL

CREATE DATABASE car_rental_db


2. В файле `database.py` укажите строку подключения к БД, например:


(логин, пароль и имя БД замените на свои).

3. При первом запуске приложение создаст таблицы на основе моделей SQLModel
через вызов функции `create_db_and_tables()` (обычно она вызывается при старте `main.py`).

### 4. Запуск серверной части (API)

uvicorn main:app --reload

По умолчанию API будет доступен по адресу:

- http://127.0.0.1:8000

Интерактивная документация Swagger UI:

- http://127.0.0.1:8000/docs

### 5. Запуск клиентского интерфейса

Откройте в браузере файл `frontend/index.html`.
